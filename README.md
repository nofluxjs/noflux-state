## Usage

All Noflux documentations can be found [here](https://github.com/nofluxjs/noflux).

You can read the online version at [noflux.js.org](https://noflux.js.org/).

## Contribution

For development:

```bash
npm install
npm run dev
```

For publish:

```bash
npm publish
```
